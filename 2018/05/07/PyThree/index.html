三周目 | 生成验证码

这次的题目要生成一个数字验证码：

【第 0010 题】

使用 Python 生成类似于下图中的字母验证码图片

如题。

需求分析

  1. 生成随机字母
  2. 生成一个 canvas 来制作图片
  3. 生成随机色块，来填充 canvas 背景
  4. 在特定区间内，随机增加字母位置的坐标，实现字母的随机间距和随机行距
  5. 高斯模糊化全图

经过这样的处理，并经过测试，腾讯提供的 OCR 是无法识别生成图片文本内容的。
